## Overview

Real-time dashboard for Sylvain venue operators to manage auditoriums, control subsystems (SPECTRA, SONORA, SENTIO, STRATUM), schedule shows, and monitor hardware health across all venue tiers.

## Features

- **Dashboard**: Live overview of all venues and auditoriums
- **Subsystem Control**: Direct control of display, audio, AI, and volumetric systems
- **Show Management**: Schedule, preview, and orchestrate screenings
- **Health Monitoring**: Real-time hardware metrics, alerts, and diagnostics
- **WebSocket Updates**: Sub-second latency for all status changes

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS (Sylvain dark theme)
- WebSocket for real-time updates
- Sylvain SDK integration

## Quick Start

```bash
npm install
npm run dev
```
