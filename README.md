## Features

- **Dashboard** — Live overview of all venues and auditoriums
- **Subsystem Control** — Direct panels for SPECTRA, SONORA, SENTIO, STRATUM
- **Show Management** — Schedule, preview, and orchestrate screenings
- **Health Monitoring** — Real-time hardware metrics, alerts, diagnostics
- **WebSocket Updates** — Sub-second latency for all status changes
- **Sylvain Dark Theme** — Black + gold/amber design language with Tailwind

<br/>

## Quick Start

```bash
npm install
npm run dev    # → http://localhost:3000
```
